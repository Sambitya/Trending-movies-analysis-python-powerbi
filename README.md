# 🎬 Trending Movies Analysis (Python + Power BI)

## 📌 Project Overview

This project analyzes **trending movies over the years** using Python for data analysis and Power BI for interactive visualization. The goal is to uncover patterns in movie popularity, release trends, and audience preferences.

---

## 🎯 Key Goals

* Understand how movie trends change over time
* Identify peak years for movie releases
* Analyze popularity and rating patterns
* Visualize insights using Power BI dashboards

---

## 🧪 Python Analysis (Jupyter Notebook)

Performed using:

* Data cleaning and preprocessing
* Exploratory Data Analysis (EDA)
* Trend identification using visualizations

---

## 📊 Power BI Dashboard

The dashboard provides:

* Year-wise movie trends
* Popularity distribution
* Comparative insights across time

---

## 📈 Key Insights

* Movie trends show cyclical growth over the years
* Certain years have significantly higher releases
* Popularity patterns highlight audience preference shifts

---

## 🛠️ Tools Used

* Python (Pandas, Matplotlib)
* Jupyter Notebook
* Power BI

---

## 📁 Files Included

* `.ipynb` → Python analysis
* `.pbix` → Power BI dashboard
* `.csv` → Dataset

---

## 🚀 How to Use

1. Open the notebook to explore analysis
2. Open `.pbix` in Power BI Desktop
3. Interact with dashboard visuals
